# 温馨提示

# 如果按照教程配置好后还是无法连接：

  1. 检查软件版本是否为最新
  2. 电脑的话清理DNS缓存后重启（管理员模式下运行 netsh winsock reset）
  3. 手机的话重启一下
  4. 还是不行就去群里问问，有可能服务器崩了
