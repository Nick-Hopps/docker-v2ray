# 使用教程

# Windows

# 所需软件

  • V2rayN（开源免费）：https://github.com/2dust/v2rayN/releases

下载 v2rayN-Core.zip 后解压到本地。

# MacOS

# 所需软件

  • V2RayX（开源免费）: https://github.com/Cenmrev/V2RayX/releases

下载 V2RayX.app.zip 后解压到本地。

# 软件配置

将文件解压出来后，找到 V2RayX 并运行，运行后软件默认在任务栏显示，不会显示主界面。

首先点击 Configure, 会弹出以下界面：

接着，依次点击 Import → import from other link, 在新弹出的窗口中，输入以下链接地址：

vmess://ew0KICAidiI6ICIyIiwNCiAgInBzIjogIldpa2lob3N0LS1ISyIsDQogICJhZGQiOiAicGF0aDJ0cnV0aC50ayIsDQogICJwb3J0IjogIjQ0MyIsDQogICJpZCI6ICJlZmI0MWZhOS03MDBkLTRjMjQtYjY4MS0yNDQ5YTQ3MWJmODkiLA0KICAiYWlkIjogIjY0IiwNCiAgIm5ldCI6ICJ0Y3AiLA0KICAidHlwZSI6ICJub25lIiwNCiAgImhvc3QiOiAiIiwNCiAgInBhdGgiOiAiIiwNCiAgInRscyI6ICJ0bHMiDQp9

输入完毕后，点击 OK 保存配置.然后，点击 server → Wikihost--HK, 确认选择该服务器。

最后，根据自己的需求，可以选择 PAC(自动)模式 或者 Global(全局)模式，然后再点击 Load core 即可开启代理。
